# Мини-Склад

Конфигурация 1С.
Прикладное мини-решение учёта склада.
Базовая интеграция с ФГИС "Меркурий".


## Стэк технологий


- справочники, документы, регистры, отчёты, печатные формы
- набор запросов и проведение по нескольким регистрам (включая FIFO)
- базовые права/роли, регламентные задания, простой HTTP-обмен
- настройка интеграции «Меркурий» в 1С 8


## Технические требования


- Платформа 8.3.21 и выше. Совместимость с 8.3.17.


## Установка и запуск


Конфигурация выгружена средствами Конфигуратора в текстовый файл .xml:

___mini-sklad.xml___

1. Создать новую Информационную базу
- файловую разместить в каталоге локально на компьютере
- серверную разместить в настроенном кластере серверов 1С, подключенном к серверу СУБД Microsoft SQL Server
- присвоить имя Мини-Склад
2. Открыть созданную Информационную базу в режиме Конфигуратора
3. Загрузить xml-файл информационной базы Мини-Склад: 
- Конфигурация -> Загрузить конфигурацию из файлов...
- выбрать XML-файлы
- Выбрать каталог и файл ___mini-sklad.dt___
4. Запустить приложение в Режиме отладки или выйти из конфигуратора и запустить Информационную базу в режиме Предприятия
